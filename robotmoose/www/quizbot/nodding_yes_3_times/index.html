Quizbot Nodding Yes 3 Times


Step 1:

Let's start off by loading our nod_yes Experiment. Type in "nod_yes" into the Experiment Name Box, and click load.


Step 2:

Rename the Experiment to "nod_yes_3".

Now, click the Run button. This created a copy of the nod_yes Experiment.


Step 3:

Click the Stop button. Go back up to the start State. Store a Variable named "times" that starts at the Value 0.


Step 4:

Add a new State. Name it "loop". Set the Run Time of the stop State to 1000 ms.


Step 5:

In the loop State, we are going to do two things:

First, add 1 to the Stored Variable times.

Second, add an If Statement that checks if the Stored Variable times is less than 3. If the Stored Variable times is less than 3, Jump to the up State.


Step 6:

Save and Run the Experiment by clicking the Run button.

The Quizbot should now center for 1000 ms, nod yes 3 times, then center.
