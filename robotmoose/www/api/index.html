RobotMoose API (Application Programming Interface)

Basic User Interface (UI)

button("string", state1)
  • string is button's title, pressing the button will run the state named state1


checkbox("string")
  • string is the checkbox's title


label("string")
  • string will appear in the UI window


print("string")
  • string will appear to the right side of the state in the code editor


slider("string", -1.0, 0.0, 1.0)
  • creates a slider in the UI titled string, with range -1.0 to 1.0 and default slider placement at 0.0


state = state1
  • runs the state named state1


store.var=j
  • stores a variable named var, initialized to value j


Create2

drive(i, j)
  • sets the speed of the left wheel to i, sets the speed of the right wheel to j (range is 0.0 to 1.0)


stop()
  • stops the robot


PWM (Pulse Width Modulation)

power.pwm[i] = j
  • sets the value of pwm[i] to j (range is 0 to 255)


Servos

power.servo[i] = j
  • moves servo[i] to position j (in degrees, range is 0 to 180)


Control Flow

if(expression)
    statement1
else
    statement2
  • if expression evaluates to true, executes statement1, otherwise executes statement2
