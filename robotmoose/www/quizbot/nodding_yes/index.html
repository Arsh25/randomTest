Quizbot Nodding Yes


Step 1:

Start by playing with the Sliders in the Drive tab. Figure out which one controls the Servo that moves the head up and down.


Step 2:

In our example, the up down Servo is on Pin 9 (or the top Slider). If yours is on the bottom Slider, you can swap it in two ways:

You could change the Hardware, and switch Pins.

Or, you could swap the order of the Servos in the Configure tab by dragging the Cross, and clicking the Configure button again, like below.

Step 3:

Since the up down Servo is Pin 9, we will swap ours back, but leave yours swapped if it is Pin 10.


Step 4:

Now we will switch to the Code tab.

Now would be a good time to resize your window so you can see all of the Code window.


Step 5:

Let's start off by setting the name of the Experiment. Let's call it nod_yes


Step 6:

Next, use the "start" State to center the Servos.

The Servo centers used on our Quizbot are 30, for the up down Servo, and 90 for the left right Servo.

These are more than likely different for your Quizbot!


Step 7:

Save and Run the Experiment by clicking the Run button.

Your Servos should now be centered!


Step 8:

Hit the Stop button. Add a new State by hitting the Add State button. Name this State "up".


Step 9:

Change the Run Time in the start State to 1000. This makes the start State run for 1000 ms (or 1 second).

Change the code in the up State to make the up down Servo move up.


Step 10:

Save and Run the Experiment by clicking the Run button.

The Quizbot should now center for 1000 ms, and then move up.


Step 11:

Hit the Stop button. Add a new State by clicking the Add State button. Call this State "down".

Copy and paste the code from the up State, and make the code move the up down Servo down.

Finally, set the Run Time for the up State to 1000 ms.


Step 12:

Save and Run the Experiment by clicking the Run button.

The Quizbot should now center for 1000 ms, move up for 1000 ms, and, finally, move down.


Step 13:

Hit the Stop button. Add one last State by clicking the Add State button. Call this State "stop".

Copy the code from the start State into this state. Set the Run Time for the down State to 1000 ms.


Step 14:

Save and Run the Experiment by clicking the Run button.

The Quizbot should now center for 1000 ms, move up for 1000 ms, move down for 1000 ms, and, finally, center.
